## GIT Basics

GIT is a powerful file version control system. It gives us a history to track progress on files and rewind or fast forward to a specific moment in time to previous versions of a file. It is kind of like the best undo button you will lever learn. GIT also allows us to make backups of our code, as well as easily share code or collaborate on code with others. GIT is the current industry standard for version control and code collaboration.

Watch the videos below if you are unfamiliar with GIT. We will be using GIT to access course materials and to share and collaborate on project code.
